# Menu Chef
## Description
The goal of the project is to create a web app that allows users to create recipes with their own ingredients.
The user can select several recipes to view their ingredients and modify them according to what the user already has.
It is also possible to record the ingredients in your refrigerator to automatically detect missing products.
